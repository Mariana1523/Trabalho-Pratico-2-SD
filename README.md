## Descrição do Projeto

Este trabalho consiste na implementação de um sistema Peer-to-Peer (P2P) simétrico utilizando Python, Threads e Sockets TCP.

O objetivo é permitir que múltiplos computadores compartilhem arquivos sem depender exclusivamente de um servidor central. Assim que um peer recebe um bloco do arquivo, ele passa a disponibilizar imediatamente esse bloco para outros peers da rede.

Dessa forma, todos os participantes colaboram tanto no download quanto no upload dos dados, caracterizando um sistema P2P descentralizado.
